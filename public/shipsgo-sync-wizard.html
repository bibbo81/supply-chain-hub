Sincronizza programmaticamente usando la nostra API:

POST /api/sync-shipsgo { "mode": "full|incremental", "types": ["awb", "container", "bl"], "dateFrom": "2024-01-01", "shipsgo_api_key": "your_key" }

Come Funziona

1
Connessione ShipsGo
Utilizziamo le tue credenziali API esistenti per accedere ai dati
2
Recupero Tracking
Scarichiamo tutti i tracking attivi (AWB) o processi la tua lista (Container)
3
Import nel Sistema
Salviamo tracking e timeline eventi nel database locale
4
Sync Continuo
Aggiornamenti automatici ogni 4-6 ore per tracking attivi
Import di Grandi Volumi: Per oltre 1000 tracking, il processo viene eseguito in background. Riceverai una notifica email al completamento con il report dettagliato.
